# Monetiza com Elas Design

Apresentacao da nova identidade visual proposta para a cliente, organizada como uma pagina estatica em portugues.

Observacao de escopo: a landing exibida na apresentacao entra como referencia de aplicacao da identidade visual e nao faz parte do servico atual. Sua execucao pode ser contratada como escopo adicional.

Alguns elementos apresentados no dashboard entram como apoio visual para demonstrar a direcao estetica da interface e nao devem ser lidos, por si so, como definicao final de funcionalidades.

## Arquivos principais

- `index.html`: apresentacao principal da proposta visual.
- `stitch/`: material original extraido do `stitch.zip`, com telas e racional criativo.

## Como visualizar

Abra `index.html` no navegador ou sirva a pasta com um servidor estatico simples.
